This is an example music directory structure for TuiTunes.

To test the application:
1. Add some MP3, WAV, or other supported audio files to this directory
2. Run: ./tuitunes example_music/

Supported formats:
- MP3
- WAV  
- FLAC
- M4A
- AAC
- OGG

The app will automatically scan subdirectories and organize tracks by artist/album based on the folder structure.
